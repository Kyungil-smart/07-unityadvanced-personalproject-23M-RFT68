### 게임을 만들면서...
- 원래는 "좀비고등학교" 게임처럼 4방향으로만 움직이고, 일정 범위안에 들어가면 좀비가 쫓아오는 로직을 만들어서 4스테이지(스테이지3 + 보스 스테이지1)의 게임을 구상하였으나... 게임을 막상 만들다보니 캐릭터 이동 로직을 하루종일 짜봤지만 w를 누르면 왼쪽으로 s를 누르면 오른쪽으로 이동하고, ad는 반응도 없는 캐릭터를 보면서 멘탈이 나갔습니다. 주말에는 개인 일정때문에 지방에 내려가서 주말동안 안절부절하면서 연휴가 끝나고 돌아와서 게임을 결국 뱀서라이크 게임으로 방향을 틀게되었고, 유니티 게임만드는 책과 유튜브 강의를 통해서 이번에는 로직짜는것(이게 왜 이렇게 구현이 되는걸까? 이건 왜 구현한거고 무슨 역할을 하는가?)을 직접 따라하면서 "내가 따라해서 만들었다!"가 아니라 이건 이렇게 구현했더니 게임이 점점 만들어지는것에 대해서 알게된거같습니다.
